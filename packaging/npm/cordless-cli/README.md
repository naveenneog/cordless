# cordless-cli

A convenience alias for **[`@naveenneog/cordless`](https://www.npmjs.com/package/@naveenneog/cordless)** —
a CLI-first remote terminal that lets you drive many Claude Code / Codex / GitHub Copilot / shell
sessions from your phone.

```bash
npx cordless-cli          # opens the dashboard + a pairing QR
# or
npm install -g cordless-cli
cordless
```

Installing this package pulls in `@naveenneog/cordless` and exposes the **`cordless`** command — it is
identical to installing the scoped package directly. See the
[full README](https://www.npmjs.com/package/@naveenneog/cordless) and the
[project site](https://naveenneog.github.io/cordless/).
